This tab is used to manage group members

After creating a group, share the group join code with others so that they can join the group. Your display name can be different for each group you are in.

Group members can click on their own name in the members list to update their display name for the group, or make themselves inactive.

Group admins can edit all group members and give or remove admin rights to other members. An admin may not remove their own admin rights - another admin must peform that update.

Group members can only be deleted if they don't have any expenses/splits assigned to them. Group members who are made inactive will no longer show up in the dropdown list when creating a new expense.
