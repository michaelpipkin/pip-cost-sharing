Ad Hoc DB Query Scripts

A small, self-contained toolbox for running ad hoc queries against the live pip-cost-sharing Firestore database and Firebase Auth. Uses the same firebase-admin SDK as the Cloud Functions, so it bypasses Firestore security rules and can query anything.

⚠ These scripts target live production data. Keep queries read-only unless you are intentionally mutating documents.

One-time setup

  1. Install Google Cloud CLI if not already present.
  2. Log in with Application Default Credentials — this is what the Admin SDK uses to authenticate: 
    gcloud auth application-default login
    A browser window opens; sign in as the GCP project owner. Credentials are stored locally at ~/.config/gcloud/application_default_credentials.json — no service-account key file is created or committed.
  3. Install dependencies (run from the repo root): 
    pnpm --dir scripts/db install

Running queries

Scripts use Node 24's built-in TypeScript stripping (--experimental-strip-types), so no external TypeScript runner is needed. Always run commands from the repo root to ensure Node resolves firebase-admin from scripts/db/node_modules.

Scratch file (ad hoc)

Edit query.ts with whatever you need, then run from the repo root:

pnpm --dir scripts/db query

Example scripts

Run any example directly from the repo root (Node walks up to find scripts/db/node_modules):

node --experimental-strip-types scripts/db/examples/users-last-7-days.ts
node --experimental-strip-types scripts/db/examples/expenses-count-aggregation.ts
node --experimental-strip-types scripts/db/examples/groups-and-members.ts

Available helpers (lib.ts)

ExportDescription
db Firestore Admin SDK instance — use exactly like db in functions/src/index.ts
auth Firebase Auth Admin SDK — access user records, listUsers(), getUser(uid), etc.
dump(snap) Pretty-print every document in a QuerySnapshot
logCount(label, n) Print a labelled count: Total users: 42

Quick-reference patterns

// Count all documents in a collection
const snap = await db.collection('users').count().get();
console.log(snap.data().count);

// Count with a filter (date strings are lexicographically comparable)
const thirtyDaysAgo = new Date(Date.now() - 30*864e5).toISOString().slice(0,10);
const recent = await db.collectionGroup('expenses')
  .where('date', '>=', thirtyDaysAgo).count().get();

// Fetch docs and print them
const usersSnap = await db.collection('users').limit(10).get();
dump(usersSnap);  // from lib.ts

// Follow a DocumentReference field
const groupDoc = await db.collection('groups').doc('someGroupId').get();
const catRef = groupDoc.data()?.defaultCategoryRef;
if (catRef) {
  const catDoc = await catRef.get();
  console.log(catDoc.data());
}

// Firebase Auth — registration time (Firestore users have no createdDate)
const { users } = await auth.listUsers(1000);
const cutoff = Date.now() - 7*864e5;
const newUsers = users.filter(u =>
  new Date(u.metadata.creationTime).getTime() >= cutoff
);
